# Ejecutar en desarrollo

1. Clonar el repositorio
2. Ejecutar

```
yarn install
```

3. Tener intalado el Nest CLI instalado

```
npx nest --version
```

4. Levantar la base de datos

```
docker compose up -d
```

5. Prisma para el manejo de la base de datos

```
npx prisma studio
```
